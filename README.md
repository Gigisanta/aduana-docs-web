# AduanaDocs Web

Landing page + market-research-informed prototype for a customs documentation management platform aimed at despachantes de aduana, agentes de transporte aduanero, forwarders, and import/export operators.

## Scope

- Single static `index.html`.
- No build step, no dependencies, no backend.
- Includes market research, ICPs, feature proposal, competitor framing, and a small interactive operations demo.

## Safety note

This prototype does **not** upload or process real customs documents. A production version needs authentication, authorization, encrypted storage, audit logs, backups, and verified legal/compliance review.

## Deploy

Static HTML deploys directly on Vercel.
